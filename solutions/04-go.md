# Lösung 04 — Go über PKCS#11

## Kernablauf

1. `pkcs11.New(modulePath)` und `Initialize()` aufrufen.
2. Slots mit Token über `GetSlotList(true)` holen.
3. Für jeden Slot `GetTokenInfo(slot)` lesen und `Label` gegen `dev-token` matchen.
4. Session öffnen, `Login(CKU_USER, pin)`.
5. Key suchen:
   - `CKA_CLASS = CKO_PRIVATE_KEY`
   - `CKA_ID = []byte{0x01}`
6. Mit `CKM_SHA256_RSA_PKCS` signieren.
7. Signatur gegen den exportierten Public Key mit OpenSSL prüfen.

## Verifikation

```bash
pkcs11-tool --module "$PKCS11_MODULE" --token-label "$PKCS11_TOKEN_LABEL" \
  --read-object --type pubkey --id 01 --output-file lab/work/public-go.der

openssl rsa -pubin -inform DER -in lab/work/public-go.der -out lab/work/public-go.pem
openssl dgst -sha256 -verify lab/work/public-go.pem \
  -signature lab/work/go.sig lab/work/go.txt
```

Erwartet: `Verified OK`.

## Typische Fehler

| Fehler | Ursache |
|---|---|
| `CKR_PIN_INCORRECT` | Falsche User-PIN beim Login |
| Kein Slot gefunden | Token-Label falsch oder Token nicht initialisiert |
| Kein Key gefunden | Falsche `CKA_ID` oder RSA-Key nicht erzeugt |

## Minimalbeispiel

`main.go`:

```go
package main

import (
	"fmt"
	"log"
	"os"
	"strings"

	"github.com/miekg/pkcs11"
)

func main() {
	module := getenv("PKCS11_MODULE", "/usr/lib/softhsm/libsofthsm2.so")
	tokenLabel := getenv("PKCS11_TOKEN_LABEL", "dev-token")
	pin := getenv("PKCS11_USER_PIN", "987654")
	keyID := []byte{0x01}

	p := pkcs11.New(module)
	if p == nil {
		log.Fatalf("module load fehlgeschlagen: %s", module)
	}
	must(p.Initialize())
	defer p.Destroy()
	defer p.Finalize()

	slot, err := findSlotByLabel(p, tokenLabel)
	must(err)

	sess, err := p.OpenSession(slot, pkcs11.CKF_SERIAL_SESSION|pkcs11.CKF_RW_SESSION)
	must(err)
	defer p.CloseSession(sess)
	must(p.Login(sess, pkcs11.CKU_USER, pin))
	defer p.Logout(sess)

	tmpl := []*pkcs11.Attribute{
		pkcs11.NewAttribute(pkcs11.CKA_CLASS, pkcs11.CKO_PRIVATE_KEY),
		pkcs11.NewAttribute(pkcs11.CKA_ID, keyID),
	}
	must(p.FindObjectsInit(sess, tmpl))
	handles, _, err := p.FindObjects(sess, 1)
	must(err)
	must(p.FindObjectsFinal(sess))
	if len(handles) == 0 {
		log.Fatalf("kein Private Key mit CKA_ID=%x", keyID)
	}

	data := []byte("hello from go pkcs11")
	mech := []*pkcs11.Mechanism{pkcs11.NewMechanism(pkcs11.CKM_SHA256_RSA_PKCS, nil)}
	must(p.SignInit(sess, mech, handles[0]))
	sig, err := p.Sign(sess, data)
	must(err)

	must(os.WriteFile("lab/work/go.txt", data, 0o644))
	must(os.WriteFile("lab/work/go.sig", sig, 0o644))
	fmt.Printf("Signatur (%d Bytes): lab/work/go.sig\n", len(sig))
}

func findSlotByLabel(p *pkcs11.Ctx, label string) (uint, error) {
	slots, err := p.GetSlotList(true)
	if err != nil {
		return 0, err
	}
	for _, s := range slots {
		info, err := p.GetTokenInfo(s)
		if err != nil {
			continue
		}
		if strings.TrimRight(info.Label, " ") == label {
			return s, nil
		}
	}
	return 0, fmt.Errorf("kein Slot mit Token-Label %q", label)
}

func getenv(k, def string) string {
	if v := os.Getenv(k); v != "" {
		return v
	}
	return def
}

func must(err error) {
	if err != nil {
		log.Fatal(err)
	}
}
```

Build und Lauf im Go-Container (`main.go` z. B. unter `lab/go-demo/main.go`):

```bash
docker compose -f lab/docker-compose.yml run --rm pkcs11-go bash
# im Container:
cd lab/go-demo
go mod init demo
go get github.com/miekg/pkcs11
go run main.go
```
